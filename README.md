# SAE1.03-04_Proxivoisin
## Projet en groupe - Plateforme de covoiturage - Langage Python - HTML - SQL

Projet de groupe à l'IUT Nord Franche-Comté

Proxivoisin est une plateforme fictive de covoiturage développée pour un projet de première année à l'IUT Nord Franche-Comté.

Pour tester cette plateforme, il faut avoir d'installé Python avec les modules flask et pymysql, ainsi que mySQL avec une base de données. <br />
Avant de lancer l'application, il faut exécuter le programme SQL dans une console mySQL et entrer les informations de connexion à la base de donnée dans le fichier app.py (lignes 15 à 18), puis lancer l'application avec le fichier launcher.sh
